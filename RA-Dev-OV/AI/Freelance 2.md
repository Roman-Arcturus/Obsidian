# Phase 0 — Foundations and Workspace Setup (4–6 weeks)

Goal: Prepare a modern, reliable development environment on Linux and Windows.

### Tasks

1. **Install Python (3.12+ or 3.13+)** on both systems.
2. **Set up environment managers**:
    - uv or pyenv (optional)
    - venv or uv venv (primary)
    
3. **Install essential tools**:    
    - Git
    - VS Code or PyCharm Community
    - Docker Desktop (Win) / Docker Engine (Linux)
    
4. **Learn Git basics**:
    - clone, commit, push, pull
    - branching
    - basic merge/PR concepts
    
5. **Terminal mastery** (bash/zsh/powershell):
    - navigation
    - pipes
    - editing commands
    - using help/man pages

### Outcome

You understand your tools and can create/manage isolated Python projects without friction.

4. Exceptions and error handling
5. Classes and OOP
6. Standard library essentials:
    - pathlib
    - datetime
    - json

7. Virtual environments and dependencies (pip/uv)
